Cada uno de vosotros debe de modificar el documento de Personas y añadir su nombre y mail.
Se debe de realizar siguiendo filosofía de GitFlow (los que venís por la mañana 

1. Debéis clonaros el proyecto
2. Crearos una rama Features para hacer vuestras cosas (al menos 2 commits, uno con vuestro nombre y otro con vuestro mail)
3. Volcar vuestra rama Feature en Develop.
4. Hacer el push a Develop.

Si conseguís visualizar vuestros datos en el repo remoto en la rama develop, lo habéis hecho bien.
Obviamente, tenéis que solventar todos los conflictos que se os presenten.

Fallos que tenéis que evitar:
· Hacer commits en la rama develop
· Subir la rama feature
· No saber qué estáis haciendo, ni por qué.
